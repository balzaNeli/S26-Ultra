# 🌌 Samsung Galaxy S26 Ultra - Landing Page

Bem-vindo ao repositório da Landing Page conceito do **Samsung Galaxy S26 Ultra**. Este projeto foi desenvolvido com foco em estética premium, alta performance e interatividade moderna, visando recriar a experiência imersiva de um lançamento de tecnologia de ponta.

🔗 **Acesse o site ao vivo:** [balzaneli-s26-ultra.vercel.app](https://balzaneli-s26-ultra.vercel.app/)

---

## ✨ Destaques do Projeto

- **Estética Premium:** Design escuro ("Dark Mode" nativo) com gradientes luxuosos, efeitos de vidro (*Glassmorphism*) e tipografia moderna que refletem o padrão da linha Galaxy.
- **Hero Imersivo:** Vídeos otimizados em loop integrados perfeitamente ao fundo da tela inicial para gerar máximo impacto visual.
- **Galeria de Cores Interativa:** Um layout em grid elegante permitindo aos usuários alternar dinamicamente entre as cores exclusivas do aparelho (Preto Titânio, Roxo Titânio, Sabor Titânio e Bege Titânio).
- **Animações Fluídas:** Integração com a biblioteca **AOS (Animate On Scroll)** para fazer os elementos surgirem suavemente conforme a rolagem da página.
- **Performance Extrema:** Vídeos comprimidos via `ffmpeg` e projeto construído com Vite, garantindo carregamento na velocidade da luz e navegação sem travamentos.

---

## 🛠️ Tecnologias Utilizadas

- **[React 18](https://reactjs.org/)** 
- **[Vite](https://vitejs.dev/)** (Ferramenta de build super rápida)
- **[Tailwind CSS](https://tailwindcss.com/)** (Para estilização ágil, responsiva e elegante)
- **[AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)** (Para as animações de rolagem da página)

## 🚀 Como executar localmente

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```
2. Entre na pasta do projeto:
```bash
cd S26-Ultra
```
3. Instale as dependências:
```bash
npm install
```
4. Inicie o servidor de desenvolvimento local:
```bash
npm run dev
```
